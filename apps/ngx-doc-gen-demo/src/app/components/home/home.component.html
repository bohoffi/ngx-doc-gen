Welcome to ngx-doc-gen

This page is for showcasing the generated API docs for the example libraries contained in the repository.

Those libraries feature two possible aspects of common library layouts - single and multi entrypoint. All results are used here as component templates.

Single entrypoint open_in_new

Maybe the most common library layout out there - where there is one single entrypoint exporting your librarys API.

Multi entrypoint open_in_new

Known from popular libraries like Angular Material - your library contains multiple entrypoints focussed on specific features.

Note: This showcase uses Angular Materials Expansion Panel to visually divide the entrypoints API docs - generation outputs one HTML file per entrypoint.

Standalone API open_in_new

Showcasing the support for the Angular Standalone API.
